Utilisation d'un relief Geomorph comme "carte de normales" Povray

Le script crack_bumptest.pov conçu en développant l'outil de génération de craquelures peut être employé avec d'autres types de terrain pour produire des résultats intéressant. Voici deux exemples.




Ce terrain ondulé a été créé de cette façon:
  1. Créer un terrain "uniforme" gris (le défaut quand vous démarrez Geomorph).
  2. Appelez l'outil "ondes" :
    1. Dans l'onglet 1 (celui avec une onde prédéfinie à la verticale), réglez l'amplitude à environ 10 et la période à 6.
    2. Dans l'onglet 2 (celui avec une onde dans l'axe nord-sud), réglez l'amplitude à environ 55, la période à 9 et donnez une rotation de 90°.
  3. Choisissez la caméra prédéfinie 3, faites une rotation de 135° dans l'aperçu.
  4. Exécutez crack_bumptest.pov.
 


Cet autre terrain a été créé de cette façon:
  1. Choisissez un terrain d' "addition progressive de surfaces" .
  2. Dans le dialogue de création et dans le sous-dialogue "Surface primitive" -> "Filtre de forme", choisissez l'image "sans filtre" comme bloc de construction.
  3. En continuant dans le dialogue de création, déroulez le sous-dialogue "Contrôle des longueurs d'ondes", initialisez toutes les longueurs d'onde à 0 (il y a un bouton unique pour cela) et réglez la longueur d'onde 64 à 50 ou 60.
  4. Exécutez  crack_bumptest.pov.
Dans le dialogue de création, vous pouvez aussi essayer différents niveaux de déplacement dans le plan XY.
