Les tutoriels et les HOW-TOs

Chaque version de Geomorph apporte des changements mineurs aux scripts Povray fournis avec le logiciel et aux options de base des dialogues. Il se pourrait que vous n'obteniez pas exactement les mêmes résultats que ceux des tutoriels. À vous de faire les adaptations nécessaires, les principes d'utilisation de base ne changent généralement pas.
